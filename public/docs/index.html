Introduction

This documentation aims to provide all the information you need to work with our API.

As you scroll, you'll see code examples for working with the API in different programming languages in the dark area to the right (or as part of the content on mobile). You can switch the language used with the tabs at the top right (or from the nav menu at the top left on mobile).

curl --request GET \
    --get "https://api-jdih.perpusnas.go.id/list-artikel?page=2&limit=3&q=hukum+islam&sort=created_at%2Cdesc" \
    --header "Content-Type: application/json" \
    --header "Accept: application/json"
const url = new URL(
    "https://api-jdih.perpusnas.go.id/list-artikel"
);

const params = {
    "page": "2",
    "limit": "3",
    "q": "hukum islam",
    "sort": "created_at,desc",
};
Object.keys(params)
    .forEach(key => url.searchParams.append(key, params[key]));

const headers = {
    "Content-Type": "application/json",
    "Accept": "application/json",
};

fetch(url, {
    method: "GET",
    headers,
}).then(response => response.json());

Query Parameters

page  integer optional  

Jika dikosongkan, maka default akan menampilkan halaman 1.

limit  integer optional  

Jumlah data yang akan ditampilkan dalam 1 halaman.

q  string optional  

Pencarian data berdasarkan query yang diinput oleh user.

sort  string optional  

Melakukan sort/pengurutan data ascending (asc) atau descending (desc) berdasarkan field yang diinginkan. Field yang dapat dipakai "title", "creator", "type", "subject", "created_at".
